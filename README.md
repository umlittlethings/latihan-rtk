# Latihan RTK CRUD

Template latihan Redux Toolkit dengan fitur CRUD lengkap untuk React.

## Installasi

1. `npm install`
2. `npm run dev`

## Fitur yang bisa kamu coba

- ✅ **Create**: Tambah post baru dengan judul dan isi
- ✅ **Read**: Fetch dan tampilkan daftar posts dari API
- ✅ **Update**: Edit judul dan isi post yang sudah ada
- ✅ **Delete**: Hapus post dengan konfirmasi
- ✅ **Refresh**: Reload data dari API
- ✅ **Error Handling**: Tangani error saat API gagal

## Latihan tambahan

1. Tambahkan fitur search/filter posts berdasarkan judul.
2. Buat pagination untuk menampilkan posts dalam halaman.
3. Tambahkan validasi form (minimal panjang karakter).
4. Simpan data ke localStorage sebagai cache.
5. Buat fitur "favorite" atau "like" untuk posts.
