# Latihan RTK

Template latihan Redux Toolkit sederhana untuk React.

## Installasi

1. `npm install`
2. `npm run dev`

## Fitur yang bisa kamu coba

- Fetch data posts dari API
- Tampilkan daftar post
- Tangani loading dan error

## Latihan tambahan

1. Ubah endpoint menjadi `users` atau `comments`.
2. Tambahkan tombol refresh untuk memanggil API lagi.
3. Tampilkan data dalam tabel atau kartu.
4. Buat filter untuk mencari judul post.
